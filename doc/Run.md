Before commencing the project deployment process, ensure that all items outlined in [that document](./Requirements.md) are prepared in advance. If everything is in order, let's get started!

Once this repository is cloned onto your computer using:
```git clone https://github.com/FAST-SUSHI-NEXTECH/DesktopApp.git```

Navigate to the directory where your project is deployed and execute ```npm i``` to update or install any missing packages.

It's possible that the terminal may display an error; if so, please follow the instructions provided in the terminal.

Once all necessary packages are installed, you can run the project using ```npm start```.
